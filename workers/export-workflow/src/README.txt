Bloodline family archive
=========================

This is a complete, self-contained export of your family's Bloodline data.
Nothing in this archive needs the internet, an account, or a server to
open — it works entirely offline, straight from wherever you extract it.

Start here
----------
Open START-HERE.html in any web browser (double-click it, or drag it into
a browser window). It's a small, read-only viewer for browsing everyone in
the family, their photos, documents, and Keepsake stories.

What's inside
-------------
  START-HERE.html          The offline viewer — open this first.
  README.txt               This file.
  manifest.json            A machine-readable record of exactly what this
                            archive contains: every file, its size, and a
                            checksum, so the archive can be verified later.
  data/tree.json           The complete, raw family tree data. This is the
                            authoritative record — everything else in this
                            archive (including the viewer) is a convenience
                            built from it, not a replacement for it.
  data/family.json         Basic identifying information about the family
                            and when this archive was generated.
  data/activity-log.json   The family's full activity history.
  photos/, documents/      Every photo and document referenced by the tree,
                            named by their original record ID.
  keepsakes/               Any generated Keepsake biography editions.
  reports/                 A plain-text and HTML summary of anything that
                            couldn't be included (for example, a photo that
                            was deleted before this archive was made) and
                            what to do about it.
  viewer/                  The code and assets START-HERE.html needs to run
                            (no other files inside viewer/ need to be
                            opened directly).

Keeping this archive safe
--------------------------
This file contains private family information, including photos and
personal details about living people. Store and share it the same way you
would any other sensitive personal document.
